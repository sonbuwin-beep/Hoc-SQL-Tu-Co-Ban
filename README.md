#### **Buổi 3: Truy vấn dữ liệu cơ bản**
Làm quen với việc lấy dữ liệu từ bảng thông qua các từ khóa:
* `SELECT`, `FROM`: Chọn cột và nguồn dữ liệu.
* `WHERE`: Bộ lọc dữ liệu.
* `BETWEEN`, `LIKE`: Truy vấn trong khoảng và tìm kiếm mẫu.
* **Toán tử so sánh:** `=`, `>`, `<`, `>=`, `<=`, `<>`.

#### **Buổi 4: Các câu lệnh truy vấn dữ liệu với SQL (Phần 1) **
Nâng cấp kỹ năng xử lý dữ liệu cơ bản:
* **Toán tử IN:** Lọc dữ liệu trong một danh sách tập hợp.
* **WHERE nâng cao:** Kết hợp nhiều điều kiện logic (`AND`, `OR`, `NOT`).
* **Thực hành:** * 01 File lý thuyết kèm bài tập rèn luyện cơ bản.
